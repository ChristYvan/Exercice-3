# Exercice 3 : Compteur de Mots en Java

## Description
Ce projet est une application Java console réalisée dans le cadre de l'UE Projet Informatique 2 (LPRGL3). L'objectif est de lire un fichier texte volumineux et de compter avec précision le nombre total de mots qu'il contient.

## Fonctionnalités
- Lecture de fichiers texte locaux.
- Analyse du contenu pour identifier les mots.
- Gestion des erreurs (fichier manquant ou illisible).
- Affichage du résultat dans la console.

## Prérequis
- **JDK 24** ou supérieur.
- **IntelliJ IDEA** (recommandé).
